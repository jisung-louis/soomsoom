# 숨숨 프로젝트 기능 명세서

이 문서는 정신 건강 관리 앱 '숨숨'의 MVP 기능 명세를 정리한 문서입니다.

## ✅ 핵심 기능 목록

1. **로그인 / 온보딩**
   - 구글/애플 계정으로 로그인
   - 간단한 앱 소개 온보딩 슬라이드 제공

2. **감정 기록**
   - 감정 선택 (예: 기쁨, 슬픔, 분노 등)
   - 텍스트로 감정 작성
   - 날짜별 기록 저장 및 확인

3. **알람 설정**
   - 원하는 시간에 알림 등록
   - 기상 미션(예: 흔들기, 문제 풀기) 설정
   - 알람 반복 설정

4. **명상 실행**
   - 호흡 명상, 배경 음악 재생
   - 명상 중 집중 상태 유지 지원

5. **감정 분석**
   - 주간/월간 감정 요약
   - 감정 변화 시각화
   - 사용자 맞춤 피드백

6. **마이페이지 / 설정**
   - 알림 설정, 앱 테마 설정
   - 사용자 정보 확인 및 로그아웃

---

로그인 → 온보딩 → 홈 → 감정 기록 → 감정 요약 → 명상 → 마이페이지
