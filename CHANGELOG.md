## [0.1.0] - 2025-08-XX

### 추가됨
- 🎉 숨숨 앱 MVP 초기 릴리스
- 🏗 React Native + Expo + TypeScript 기반 프로젝트 구조
- 🧭 React Navigation을 사용한 탭 기반 네비게이션
- 🎨 커스텀 UI 컴포넌트 시스템
- 📊 Zustand를 사용한 상태 관리
- 🎵 Expo AV를 사용한 오디오 재생 기능
- 🔔 Expo Notifications을 사용한 알림 시스템
- 📱 iOS 및 Android 크로스 플랫폼 지원

### 주요 기능
- **감정 기록**: 일일 감정 상태를 캘린더에 기록하고 분석
- **명상 플레이어**: 다양한 명상 콘텐츠와 호흡 운동 가이드
- **알람 시스템**: 개인 맞춤 알람 설정 및 알림
- **성취 시스템**: 목표 달성 배지와 개인 룸 꾸미기
- **사용자 프로필**: 개인 정보 관리 및 설정

### 기술적 특징
- TypeScript를 사용한 타입 안전성
- 컴포넌트 기반 아키텍처
- 재사용 가능한 UI 컴포넌트
- 커스텀 훅을 통한 로직 분리
- 반응형 디자인 및 모바일 최적화
